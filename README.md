# chrome standalone repo

This repository is for downloading offline standalone chrome.By default, when we download Chrome,the package is just an only small web installer, and need us to download the real package online.But some times, we got only offline environment, and need to install Chrome,so we can download the full Chrome offline package from following url,and use them to install with no extra online downloading.

## Stable Channel
- [Windows](https://www.google.com/chrome/browser/?platform=win64&standalone=1)
- [Mac](https://www.google.com/chrome/browser/?platform=win64&standalone=1)

## Beta Channel
- [Windows](https://www.google.com/chrome/browser/beta.html?platform=win64&standalone=1)
- [Mac](https://www.google.com/chrome/browser/beta.html?platform=mac&extra=betachannel&standalone=1)

## Dev Channel
- [Windows](https://www.google.com/chrome/browser/index.html?extra=devchannel&platform=win64&standalone=1)
- [Mac](https://www.google.com/chrome/browser/?platform=mac&extra=devchannel&standalone=1)
